# 🐶 Oskar Beach Stories

Ein kleines Browser-Spiel mit 4 Levels, entwickelt zum Programmieren lernen.

## Levels
- **Level 1** – Klick-Spiel: Gib Oskar 5 Snacks
- **Level 2** – Labyrinth: Finde den Weg zum Strand
- **Level 3** – Runner: Spring über Hindernisse
- **Level 4** – Match 3: Kombiniere 3 gleiche Emojis

## Starten
Einfach `index.html` im Browser öffnen.

## Technologien
- HTML, CSS, JavaScript (Vanilla)
- Google Fonts (Poppins)

## Assets
Eigene Illustrationen im Ordner `/assets`.
